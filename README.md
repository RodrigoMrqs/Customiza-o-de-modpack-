# Customização-de-modpack-PT-BR
Este repositorio foi criado com a ideia de criar-se um ambiente de trabalho para que multiplos programadores desenvolvam customizacoes para um modpack de minecraft para um servidor chamado Mortysium SMP com um tema pos apocaliptico, contando com integracoes entre mods e receitas alteradas, alem de outros recursos facilitando o trabalho, como funcoes para nos ajudar a criar receitas, otimizando codigo e reduzindo seu tamanho.

Pensado e criado por: Arthur "Bode" Bonikovski e Rodrigo "Shade" Marques em colaboracao com outros membros do servidor.
# Modpack-customization-ENG
This repository was created with the thought of creating a workplace where multiple devs can create customizations for a minecraft modpack to a server called Mortysium SMP, with a post-apocalyptic theme, counting with integrations between mods and altered recipes, in addition there's also some resources to make our job easier, one example being functions to help us create recipes, optimizing code and reducing its length.

Thought and created by: Arthur "Bode" Bonikovski e Rodrigo "Shade" Marques with collaboration of other members of the server.
